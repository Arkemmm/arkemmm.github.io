Les bases de l'intégration

Exercice 1 : Forme polynomiale ★★☆☆☆

Calculer l'intégrale :

$$\int_{-1}^{2}(5x^{3}-3x+4) dx$$

$$\int_{-1}^{2} (5x^3 - 3x + 4) \, dx = \left[\frac{5x^4}{4} - \frac{3x^2}{2} + 4x\right]_{-1}^{2} = \left(\frac{5(2)^4}{4} - \frac{3(2)^2}{2} + 4(2)\right) - \left(\frac{5(-1)^4}{4} - \frac{3(-1)^2}{2} + 4(-1)\right)$$

$$\int_{-1}^{2} (5x^3 - 3x + 4) \, dx = \frac{105}{4}$$

Exercice 2 : Forme trigonométrique ★★★☆☆

Calculer l'intégrale :

$$\int_{-\pi}^{\frac{\pi}{2}} \cos(2\theta-\pi) - \sin\left(\theta-\frac{\pi}{4}\right) d\theta$$

$$\int_{-\pi}^{\frac{\pi}{2}} \cos(2\theta-\pi) - \sin\left(\theta-\frac{\pi}{4}\right) \, d\theta$$

$$= \left[\frac{\sin(2\theta - \pi)}{2} + \cos\left(\theta - \frac{\pi}{4}\right)\right]_{-\pi}^{\frac{\pi}{2}}$$

$$= \left(\frac{\sin\left(2\left(\frac{\pi}{2}\right) - \pi\right)}{2} + \cos\left(\frac{\pi}{2} - \frac{\pi}{4}\right)\right) - \left(\frac{\sin\left(2(-\pi) - \pi\right)}{2} + \cos\left(-\pi - \frac{\pi}{4}\right)\right)$$

$$= \left(\frac{\sin(0)}{2} + \cos\left(\frac{\pi}{4}\right)\right) - \left(\frac{\sin(-3\pi)}{2} + \cos\left(-\frac{5\pi}{4}\right)\right) = \left(0 + \frac{\sqrt{2}}{2}\right) - \left(0 - \frac{\sqrt{2}}{2}\right)$$

$$= \frac{2\sqrt{2}}{2}= \sqrt{2}$$

Exercice 3 : Suivre son intuition ★★★☆☆

Calculer l'intégrale :

$$\int_{0}^{1}\frac{2x+2}{x^{2}+1}dx$$

$$\int_{0}^{1}\frac{2x+2}{x^{2}+1} \, dx = \left[\ln(x^2+1) + 2\arctan(x)\right]_{0}^{1} = \ln(2) + 2\arctan(1) - \ln(1) - 2\arctan(0) = \ln(2) + \frac{\pi}{2}$$

Exercice 4

Calculer l'intégrale :

$$\int_{-1}^{1}\frac{-14}{(5x+9)^{2}}dx$$

$$\int_{-1}^{1}\frac{-14}{(5x+9)^{2}} \, dx = \left[\frac{14}{5x+9}\right]_{-1}^{1} = \frac{14}{5(1)+9} - \frac{14}{5(-1)+9} = \frac{14}{14} - \frac{14}{4} = 1 - \frac{7}{2} = -\frac{1}{2}$$

Exercice 5

Calculer l'intégrale :

$$\int_{-1}^{1}\frac{2x-\pi}{\sqrt{1-x^{2}}}dx$$

$$\int_{-1}^{1} \frac{2x-\pi}{\sqrt{1-x^{2}}} \, dx = \int_{-1}^{1} \frac{2x}{1-x^{2}}+\pi \frac{-1}{\sqrt{1-x^{2}}} \, dx$$

$$= \int_{-1}^{1} \frac{2x}{\sqrt{1-x^{2}}}+ \pi \int_{-1}^{1} \frac{-1}{\sqrt{1-x^{2}}} \, dx = [-2\sqrt{1-x^2}]_{-1}^{1} + \pi [\arccos(x)]_{-1}^{1}$$

$$= -2[\sqrt{1-x^2}]_{-1}^{1} + \pi (\arccos(1) - \arccos(-1)) = 0 + \pi \cdot (- \pi )= - \pi^{2}$$

Exercice 6 : Identité trigonométrique ★★★★☆

Calculer l'intégrale :

$$\int_{-1}^{1}\frac{2x-\pi}{\sqrt{1-x^{2}}}dx$$

Pour trouver le développement limité de \(f(x)\) à l'ordre 3 en 0, nous développons le numérateur à l'ordre 4 puisque nous divisons par \(x\) ensuite :

Nous rappellons le développement à l'ordre 4 de \((1+x)^{\alpha }\) : $$(1+x)^{\alpha } = 1 + \alpha x + \frac{\alpha (\alpha - 1)}{2!} x^2 + \frac{\alpha (\alpha - 1)(\alpha - 2)}{3!} x^3 + \frac{\alpha (\alpha - 1)(\alpha - 2)(\alpha - 3)}{4!} x^4 + o(x^4)$$

Exercice 7 : Identité trigonométrique ★★★★☆

Calculer l'intégrale :

$$\int (3\sin(20x) \cos(23x) + 20 \sin(43x))dx$$

$$\int (3\sin(20x) \cos(23x) + 20 \sin(43x))dx = 3 \int (\sin(20x) \cos(23x))dx + 20 \int \sin(43x)dx$$

Nous rappelons que : \(\sin(\theta) \cos(\varphi) = \frac{1}{2} [\sin(\theta + \varphi) + \sin(\theta - \varphi)]\), ainsi nous avons :

$$ 3 \int \frac{1}{2}(\sin(43x) + \sin(-3x))dx + 20 \int \sin(43x)dx = \frac{3}{2}(-\frac{1}{43} \cos(43x) -\frac{1}{-3} \cos(-3x)) - 20 \times \frac{1}{43}cos(43x) + C $$

$$ = -\frac{3}{86} \cos(43x) + \frac{3}{6} \cos(-3x) - \frac{20}{43}cos(43x) + C = \frac{-3 - 40}{86}cos(43x) + \frac{1}{2} \cos(-3x) + C = -\frac{1}{2}cos(43x) + \frac{1}{2} \cos(-3x) + C$$

Changements de variables

Exercice 1 : Introduction à la substitution ★☆☆☆☆

Calculer l'intégrale :

$$\int\frac{x^{3}}{1+x^{4}}dx$$

Nous aurions pu traiter cet exercice avec la forme usuelle : \(\int\frac{u'}{u} = ln(u)\). Mais nous allons voir la méthode de la substitution :

Soit \(u = (1 + x^{4})\) et donc la dérivé de u s'écrit \(du = 4x^{3}dx\) ainsi nous avons : \(dx = \frac{du}{x^{3}}\), nous pouvons alors écrire :

$$\int\frac{x^{3}}{1+x^{4}}dx = \int\frac{x^{3}}{u} \times \frac{du}{4x^{3}} = \frac{1}{4}\int \frac{1}{u}du = \frac{1}{4}ln(u) + C = \frac{1}{4}ln(1 + x^{4}) + C$$

Exercice 2 : Démonstration de l'utilité de la substitution ★★☆☆☆

Calculer l'intégrale :

$$\int\frac{x}{1+x^{4}}dx$$

Nous faisons face à un cas un peu plus difficile à permière vue. Nous allons donc découvrir l'intérer de la substitution :

Soit \(u = x^{2}\) et donc \(du = 2x dx\) ainsi nous avons : \(dx = \frac{du}{2x}\), nous pouvons alors écrire :

\(\int\frac{x}{1+x^{4}}dx = \int\frac{x}{1+u^{2}} \times \frac{du}{2x} = \frac{1}{2}\int \frac{1}{1+u^{2}}du = \frac{1}{2}\arctan(u) + C = \frac{1}{2}\arctan(x^{2}) + C\)
